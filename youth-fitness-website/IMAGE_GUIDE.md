# 满分虎网站文章配图指南

## 配图原则

1. **真实性优先**：优先使用满分虎实际训练场景照片
2. **匹配度检查**：每篇文章发布前必须检查配图与内容是否匹配
3. **避免通用图**：避免使用过于通用的健身图片，要有针对性

---

## 文章主题与配图对照表

| 文章类型 | 建议配图内容 | 禁用配图 |
|---------|-------------|---------|
| 跑步训练 | 学生在跑道上跑步、田径训练场景 | 健身房举重、瑜伽 |
| 立定跳远 | 跳远动作分解、沙坑训练 | 游泳、骑行 |
| 引体向上 | 单杠训练、上肢力量练习 | 跑步机、深蹲 |
| 实心球 | 投掷动作、力量训练 | 跑步、跳绳 |
| 运动损伤 | 康复训练、拉伸指导 | 比赛冲刺、剧烈运动 |
| 家长指导 | 家长观看训练、亲子运动 | 纯成人健身 |
| 机构选择 | 教练指导学生、小班训练 | 空旷场地、无教练 |
| 考试心理 | 学生专注状态、考前准备 | 比赛庆祝、轻松休闲 |
| 寒假集训 | 冬季训练、室内体能课 | 夏季户外、游泳 |

## 配图来源建议

### 优先级1：真实照片（推荐）
- 满分虎训练基地实拍
- 学员训练场景（需家长授权）
- 教练示范动作

### 优先级2：专业图库
- Unsplash（需筛选）
- Pexels（需筛选）
- Pixabay（需筛选）

### 优先级3：关键词搜索
使用Unsplash Source API：
```
https://source.unsplash.com/800x400/?关键词1,关键词2
```

## 配图检查清单（发布前必查）

- [ ] 图片内容与文章主题是否直接相关？
- [ ] 图片中人物年龄是否为青少年/学生？
- [ ] 图片是否为体育/运动场景？
- [ ] 图片alt文本是否准确描述了图片内容？
- [ ] 图片是否有版权风险？

## 发布流程

1. **生成文章内容** → 检查质量（≥500字，软性广告植入）
2. **选择配图** → 对照本文档检查匹配度
3. **填写alt文本** → 准确描述图片内容
4. **发布前审核** → 确认内容和配图一致
5. **推送上线**
