## 图片点击跳转 + 编码问题修复 ✅

### 时间：2026-05-08 20:05-20:07

### 项目：满分虎体育网站 (youth-fitness-website / manfenhu.com)

### 问题发现
1. **点击列表页图片无法跳转** — `<div class="news-image">` 无法点击
2. **文章详情页 404** — 所有13篇文章返回 404 NOT_FOUND
3. **图片没更新** — 用户反馈图片显示异常

### 根本原因分析
- **问题1**：HTML 结构错误，用了 `<div>` 而非 `<a>` 标签
- **问题2（关键）**：文章文件被保存为 **GBK 编码**，而 Astro/Vercel 构建系统需要 **UTF-8**。GBK 文件解析失败 → 页面无法生成 → 404
- **问题3**：Vercel 部署未完成 + 浏览器缓存

### 解决方案
1. **index.astro 图片链接**：将 `<div class="news-image">` 改为 `<a class="news-image" href="/news/xxx">`，并添加 CSS hover 效果（图片放大）
2. **13篇文章转 UTF-8**：用 Python 脚本将所有文章从 GBK 转换为 UTF-8 编码
3. **推送代码**：全部修改推送到 GitHub，等待 Vercel 部署
